## 📌 项目简介

基于 **10万+ 条电商真实订单数据**，独立完成从数据清洗、特征工程到多维分析与可视化的完整链路。通过 Pandas 进行数据聚合与时间序列处理，使用 Plotly + Streamlit 构建交互式 BI 看板，并利用 MySQL 进行复杂 SQL 分析与用户留存/分层挖掘，为运营策略提供数据支撑。

## 📊 数据分析内容

### 1. 数据清洗 (`01_data_cleaning.ipynb`)
- 读取原始 Excel 数据
- 处理缺失值、异常值
- 数据类型转换
- 生成清洗后数据集 `cleaned_orders.csv`
- 📌 **结论**：原始数据中仅 2.0% 为异常记录，数据采集系统整体规范，清洗后 100,286 条数据可支撑后续多维分析。

### 2. 销售额分析 (`02_sales_analysis.ipynb`)
- 总销售额统计
- 各平台销售额对比
- 订单数量与金额关系
- 📌 **结论**：移动端（APP+公众号）贡献超九成销售额，说明用户消费习惯已完全迁移至移动端，Web 端仅作为辅助渠道存在，渠道质量存在显著差异。

### 3. 时间维度分析 (`03_time_analysis.ipynb`)
- **每日销售趋势**：追踪全年日销变化，发现销售周期规律
- **每小时订单分布**：分析用户下单高峰期，优化运营策略
- **星期几订单分布**：识别周末/工作日消费差异
- 📌 **结论**：销售额集中在特定时段和周末，说明平台存在明显活跃高峰和周期性消费规律，可据此优化促销排期和客服排班策略。

### 4. 商品与用户分析 (`04_product_user_analysis.ipynb`)
- **商品销售额 TOP10**：识别爆款商品，建立库存预警机制
- **用户消费 TOP10**：挖掘高价值用户，支撑会员营销
- 📌 **结论**：TOP10 商品贡献 35% 销售额、TOP10 用户贡献超 18 万元，少数爆款和头部用户驱动了大部分业绩，验证了电商典型的"二八定律"特征。

### 5. 可视化看板 (`05_visualization.ipynb`)
- 4 张独立分析图 + 1 张四合一总览图
- 柱状图带数值标签
- 7日移动平均线平滑处理
- 饼图占比优化（合并小渠道）
- 📌 **结论**：四合一业务看板将销售、渠道、商品、用户四大维度整合为一张决策视图，管理层可在一屏内快速完成日常经营诊断。

### 6. SQL 高级分析 (`sql/`)
- **留存分析**：次日留存率、7日/30日留存率，衡量用户粘性
- **RFM 用户分层**：基于 Recency（最近消费）、Frequency（消费频次）、Monetary（消费金额）三维度划分用户价值层级
- **用户群体分类**：重要价值用户、重要发展用户、重要保持用户、流失用户、一般用户
- 📌 **结论**：用户平均仅消费 1.31 次，近四分之一用户（22.98%）已进入流失状态，复购激活和流失预警是当前最紧迫的运营课题。

### 7. 交互式 BI 看板 (`app/bi_dashboard.py`)
- **核心指标卡**：总销售额、订单数、活跃用户、客单价、复购率、退款率
- **每日销售趋势**：折线图 + 峰值标注
- **平台销售占比**：动态饼图（单平台时自动隐藏）
- **24小时下单分布**：柱状图，揭示用户行为规律
- **RFM 用户分层**：饼图 + 业务洞察建议
- **商品/用户 TOP10**：排行榜 + 悬停详情
- **交互筛选**：平台多选、日期范围选择
- 📌 **亮点**：Toast 提示、空数据处理、动态布局、业务洞察文案

## 🎯 核心业务洞察（含真实数据）

### 1️⃣ 头部商品效应显著
- TOP10 商品单款销售额均超 **30 万元**，最高达 **48 万元**
- 前 10 商品贡献了约 **35%** 的总销售额（二八定律明显）
- 💡 建议：建立爆款库存预警机制，对 TOP10 商品设置安全库存阈值

### 2️⃣ 渠道双极化格局
| 渠道 | 销售额占比 | 订单量占比 | 客单价 |
|------|-----------|-----------|--------|
| APP | 45.3% | 44.1% | 1,320元 |
| 微信公众号 | 46.3% | 47.2% | 1,285元 |
| Web网站 | 6.6% | 7.1% | 1,190元 |
| 其他 | 1.9% | 1.6% | 1,450元 |

- 📌 移动端（APP+公众号）贡献 **91.6%** 销售额
- 💡 建议：收缩 Web 端运营资源，聚焦移动端转化优化

### 3️⃣ 销售趋势健康增长
- 原始日销波动大（10万~50万），但 **7日移动平均线** 显示稳步上升
- 全年销售额从 **20万/日** 攀升至 **30万+/日**，增幅 50%+
- 💡 建议：在波动低谷期（如周五）设计"周末预热"活动拉升

### 4️⃣ RFM 用户分层（78,060 名用户）
| 用户分层 | 人数 | 占比 | 平均消费 | 运营策略 |
|---------|------|------|---------|---------|
| 重要价值用户 | 356 | 0.46% | 4,584元 | VIP专属服务 |
| 重要发展用户 | 2,191 | 2.81% | 733元 | 凑单推荐提客单 |
| 重要保持用户 | 25 | 0.03% | 4,142元 | 大额券定向召回 |
| 流失用户 | 17,941 | 22.98% | 486元 | 低成本短信激活 |
| 一般用户 | 57,547 | 73.72% | 1,559元 | 首单复购券引导 |

- 🔑 关键发现：0.46% 的头部用户客单价是一般用户的 **3倍**
- 💡 建议：对"重要发展用户"设计"满200减30"活动，推动向价值用户转化
